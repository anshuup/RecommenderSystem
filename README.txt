Instructions for running the page on server
-> use npm install to install the required modules
-> Then node a2UserBasedServer.js to run for user based filtering
-> Then node a2ItemBasedServer.js to run for item based filtering


Files Included
a2UserBasedServer.js  -> server side script to calculate the MAE for user-based filtering
a2ItemBasedServer.js  -> server side script to calculate the MAE for item-based filtering

assignment2-ratings-data.txt    -> Contains the data for which MAE needs to be calculated
